FRCComponents
=============
 
 DXF vector drawings of FRC electircal parts for rapid prototyping electrical boards.
 
 Includes:
 
  * cRio
  
  * victor
  
  * digital side cars
  
  * Power Board
  
  * Inverter
  
  * Power Swithes
  
  * RaspberryPi
  
  * Arduino
  
[This is a great resource](http://www.firstcadlibrary.com/pages/electrical%20parts.asp)
